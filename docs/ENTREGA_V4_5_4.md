# En Redes Andamos ERP/CRM - V4.5 Entrega 4

## Objetivo
Redondear el uso diario del planner y del calendario para que el equipo pueda planificar campañas, tareas, reuniones, entregas y vencimientos con una experiencia más visual.

## Cambios principales

### Planner colaborativo
- Se agregó creación de proyectos desde el planner.
- Se reforzó la jerarquía Proyecto -> Tareas.
- Se agregaron filtros por proyecto.
- Cada proyecto muestra tareas en Kanban por estado.
- Cada tarjeta de proyecto muestra cantidad de tareas, etapas y avance promedio.

### Calendario operativo
- Se reemplazó la lista larga por una grilla mensual real.
- Selector de mes y año.
- Navegación por mes anterior, mes siguiente y hoy.
- Eventos/tareas/publicaciones como pegatinas visuales.
- Colores según tipo de evento, usando la identidad visual de En Redes Andamos.
- Hover con estado y prioridad.
- Click en tareas para abrir el detalle en planner.
- Filtros por cliente, proyecto, responsable y tipo.
